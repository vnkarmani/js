Here are 3 ways to create a variable in javascript.

  1. var:

    2. we can use var keyword to create a variable.

    using this keyword we can change its value nd redeclare it.

  2. let:

    3. we can use let keyword to create a variable.

    using this keyword we can change its value but we cant redeclare it.

  3. const:

    4. we can use const keyword to create a variable.

    once we declare it,we cant change the value of const variable's value through out the whole program.
